SIS 2 GEIPជាឧបករណ៍មួយដែលបង្កើតឡើងសម្រាប់លោកគ្រូអ្នកគ្រូដែលធ្វើការបញ្ចូលពិន្ទុក្នុងកម្មវិធី SIS និងនាំចេញមកកាន់កម្មវិធី GEIP ។ ឧបករណ៍នេះជួយសម្រួលដល់លោកគ្រូអ្នកគ្រូក្នុងការកែសម្រួលទិន្នន័យមុនពេលនាំចូលទៅកាន់ GEIP ព្រោះកម្មវិធីទាំងពីរនេះមានជួរឈរ (Columns) ខុសគ្នា ឱ្យបានត្រឹមត្រូវដើម្បីងាយស្រួល និងចំណេញពេលវេលា។

របៀបប្រើប្រាស់៖

  1. ចុចលើប៊ូតុង «ចាប់ផ្ដើម»
  2. បញ្ចូល​ (Upload) ឯកសារ Excel ដែលបាននាំចេញពី SIS
  3. ចុចប៊ូតុង «បន្ទាប់»
  4. ក្នុងករណីដែលឈ្មោះសិស្សក្នុងឯកសារដែលទាញចេញពី​ SIS ខុសលំដាប់ពីឈ្មោះដែលមានក្នុង GEIP លោកគ្រូអ្នកគ្រូត្រូវចម្លង (Copy) ឈ្មោះចេញពី GEIP និងបិទភ្ជាប់ (Paste) តាមផ្ទាំង (Tab) នីមួយ។ (បើសិនជាដូចគ្នាហើយលោកគ្រូអ្នកគ្រូអាចរំលងចំណុចនេះបាន)
  5. ចុចលើប៊ូតុង «ទាញយក»
  6. រងចាំបន្តិចឯកសារនឹងទាញយកដោយស្វ័យប្រវត្តិ
